Program działa na Python 3.7, bo wyżej nie obsługuje OpenCV, posiada szczątkowe GUI, także najlepiej go obsługiwać z wiersza poleceń.

Do działania programu wymagane jest stabilne łącze internetowe (najlepiej jak najszybsze), silnik Tesseract oraz biblioteki Pythonowe:

pytesseract
opencv-contrib-python
PythonVideoConverter
youtube-dl
alive_progress

Użytkownik może je zainstalować sam, jak również instalacja pakietów jest zaimplementowana jako opcja w samym programie. Dla systemu macOS jest zaimplementowana również opcjonalna instalacja/aktualizacja Tesseract.
